# 作業メモ (Working Memos)

研究開発中のメモ、アイデアの断片、検証ログを時系列で保存します。

## 運用ルール

1. **ファイル名規則**: `YYYY-MM-DD_nn_Title.md`
   - `YYYY-MM-DD`: 日付
   - `nn`: その日の連番（01, 02...）
   - `Title`: 内容の要約（英数字推奨）

2. **内容**:
   - 形式は問わない
   - 結論が出ていなくても良い
   - 後に体系化されたドキュメントへ昇格させる素材とする

3. **テンプレート**: `YYYY-MM-DD_nn_template.md` を使用のこと
